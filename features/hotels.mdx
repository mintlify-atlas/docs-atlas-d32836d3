## Current workarounds

While hotel search is being built, you can:

1. Use the **itinerary planner** to add accommodation entries with:
   - Hotel name and address
   - Check-in and check-out times
   - Booking confirmation number
   - Reference links to your booking

2. Add notes in the **trip overview** with hotel details

3. Use the **finance tracker** to log accommodation costs
